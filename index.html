Law Firm Executive Boardroom

Project Overview

Designed and deployed a complete AV system for a national law firm relocating to a premier NYC office. The flagship boardroom required a high-performance environment that supports Microsoft Teams video conferencing, in-room presentations, and all-hands meetings for up to 20 participants.

Technical Scope

The room features dual 86" commercial displays driven by a Crestron NVX-based AV-over-IP distribution system. A Crestron touch panel provides one-touch meeting join and source switching. Outcome

The firm's attorneys and staff can join Teams meetings, share content wirelessly, and present to clients without any technical friction. The system supports both in-room and hybrid collaboration with consistent audio/video quality.

Berlin Lions Club Event Network Infrastructure

Project Overview

As Assistant Superintendent of Technology for the Berlin Lions Club, I designed and deployed an enterprise-scale network infrastructure supporting mission-critical operations for one of Connecticut's largest annual fairs, serving 300,000+ attendees over a 10-day run.

Technical Scope

AirPlay Relay Controller

Project Overview

Built a Dockerized IoT application that turns a Raspberry Pi into an AirPlay audio receiver with automatic amplifier power management. When an Apple device starts streaming audio, the system detects the AirPlay session and triggers a GPIO relay to power on connected amplifiers.

Technical Highlights

The system uses shairport-sync (v4.3.2) for AirPlay reception with ALSA audio output and SoX resampling. A metadata pipe monitors playback state changes and a custom Bash script manages GPIO relay switching with debounce logic. The entire stack runs in a Docker container on BalenaOS.
